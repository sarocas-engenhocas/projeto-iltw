—SOBRE—

O projeto PipocaDigital foi desenvolvido no âmbito da unidade curricular Introdução às Linguagens e Tecnologias Web (ILTW), com o objetivo de criar uma aplicação web funcional, moderna e responsiva. A plataforma apresenta‑se como um catálogo de filmes, permitindo aos utilizadores explorar títulos, produtoras, géneros e outras informações relevantes de forma simples e intuitiva.

A construção do site recorreu a HTML5 e CSS3 avançado, com especial foco em Flexbox e design responsivo, garantindo compatibilidade com dispositivos móveis. A componente interativa foi implementada em JavaScript, permitindo filtrar e ordenar filmes dinamicamente, proporcionando uma experiência de navegação mais rica.

Para a comunicação de dados, o projeto utiliza a função fetch() para carregar informação a partir de um ficheiro dados.json, criado especificamente para este trabalho. Esta abordagem simula a interação com uma API real e reforça a importância da separação entre interface e dados.

O projeto integra ainda uma componente de back‑end simples, responsável por processar um formulário (como contacto ou registo) e devolver uma resposta personalizada ao utilizador, demonstrando a ligação entre o front‑end e o servidor.

O resultado final é um website organizado, funcional e visualmente apelativo, que cumpre todos os requisitos da disciplina e demonstra a aplicação prática dos conhecimentos adquiridos ao longo do semestre.
